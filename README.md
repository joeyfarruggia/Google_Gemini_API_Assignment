# Google_Gemini_API_Assignment

## Description

This project demonstrates how to use Google's Gemini API in a Python
application. The application accepts a question from the user, sends
the prompt to the Gemini 3.5 Flash model, and displays the AI-generated
response in the terminal.

## API Used

This project uses the Google Gemini API through Google's official
`google-genai` Python SDK.

## Features

- Accepts prompts through the command line
- Sends prompts to the Gemini API
- Displays AI-generated responses
- Handles missing API keys
- Handles unsuccessful API requests
- Keeps the API key outside of the source code
